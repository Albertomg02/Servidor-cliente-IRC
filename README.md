# Servidor-cliente-IRC
Servidor y cliente  para el protocolo IRC

#Compilacion
Se ejecuta make y se encarga de crear los ejecutables del servidor y cliente IRC,
y tambien del servidor y cliente echo ssl.

#Servidor
./server p <puerto>
ej: ./server p 6667

#Cliente
./client
